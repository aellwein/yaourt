Auto-install yaourt (Archlinux AUR helper)
------------------------------------------

**via curl**
```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/aellwein/yaourt/master/install.sh)"
```

**via wget**
```
sh -c "$(wget https://raw.githubusercontent.com/aellwein/yaourt/master/install.sh -O -)"
```

Enjoy!
